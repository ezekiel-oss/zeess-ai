A Copilot-style chatbot built with Next.js, TailwindCSS, OpenAI, and DeepSeek.  
Deployed on Vercel with a custom UI theme.

## Features
- Copilot-inspired UI with custom colors
- Smart routing between OpenAI and DeepSeek
- Error handling and environment variable setup
- Ready for deployment on Vercel

## Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/ezekiel-oss/zeess-ai.git
   cd zeess-ai
Install dependencies:
bash
npm install
Add environment variables in .env.local:
Code
OPENAI_API_KEY=your_openai_key_here
DEEPSEEK_API_KEY=your_deepseek_key_here
Run locally:
bash
npm run dev
Deploy on Vercel.
